Skrrt Project

A Flicker-Like Clone

The Skrrt Project was modeled after Flicker. This website encourages users to upload pictures of their cars or cars that they deam worthy of skrrt upload. Using Amazon Web Services and React, users can upload a picture and store the picture, title, description, and tag in state.

Users can also scroll through pages of pictures that other users have uploaded to the site. Users are able to do this by clicking on different numnbers that take you to differnt pages on the site. This feature was created using reacts state feature, to keep track of what page the user is currently on.

Users can also click on an image and view more information on that image. In here, users can view the title, description, and the tag it owns. If a user is the owner of the picture this is where they can make edits to the image content. If they are not the user however, they do have the option to comment on the picture. Users can also view all the comments that other users have created for that image itself.

This website was built using Express Javascript and React Redux. This site also has its own customized database to store users, images and their content, aswell as store the comments on pictures.
