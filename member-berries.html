Member-Berries

A Remember The Milk Clone

Live Site

Inspired by the app Remember The Milk, Member-Berries is a website for users to come and manage their lists and tasks to stay organized and efficient. This app allows you to create lists and add tasks to those lists.

When you securely log in to the app, you are immediately presented a clean interface where you can see all the lists that you have. These lists have a clickable uncompleted tasks section. When clicked on, a modal with all the uncompleted tasks in that list are presented to you. Inside this element a user can click a checkbox which moves the task to the completed section.

Much like the original website, Remember The Milk, this website allows you to create a list item and add tasks to those lists. This app also allows users to search for keywords so that users can easily find the task they were loking for.

This App was created using Flask for the backend and React Redux for the frontEnd. This app also has its own customized backend database to store users lists and tasks.
